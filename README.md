## **Getting Started**

1. Clone the repository:

   ```sh
   git clone https://github.com/ericelric/stylesort.git
   cd stylesort
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Start the development server:

   ```sh
   npm run dev
   ```

## Usage

1. Paste your **CSS** into the input area.
